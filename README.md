# danurrio.github.io

Portafolio personal desarrollado como carta de presentación para mi búsqueda de empleo en el ámbito de la ciberseguridad, orientado a posiciones de **SOC Analyst**.

Recoge mis proyectos, habilidades técnicas y formación de una forma visual y directa, pensada tanto para reclutadores como para equipos técnicos.

## Inspiración

Diseño basado en [andrijaweb.vercel.app](https://andrijaweb.vercel.app/), adaptado a una estética de terminal y entorno de seguridad.

## Stack

HTML · CSS · JavaScript vanilla — sin frameworks, sin dependencias.

## Contenido

- Presentación y estado de disponibilidad
- Stack de herramientas de seguridad
- Proyectos destacados: [StellaMontis](https://github.com/Danurrio/TFM-StellaMontis) y [OpenDrive](https://github.com/Danurrio/TFM-OpenDrive)
- Experiencia y formación
- Contacto
